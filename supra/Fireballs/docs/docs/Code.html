cyscan(supra_pos, detec_pos, z_profile,
wind=True, n_theta=45, n_phi=90, precision=1e-25, tol=1):

Calculates time of arrival between two points in 3-D space with an atmosphere.

Parameters

  • supra_pos (ndarray) – Local coordinates of the higher point (supracenter)

  • detec_pos (ndarray) – Local coordinates of the lower point (detector)

  • z_profile (ndarray) – 2-D slice of the atmosphere between supra_pos and detec_pos

  • wind (Boolean) – Switch to enable winds in z_profile.

  • n_theta (int) – Angle grid spacing of the zenith angle

  • n_phi (int) – Angle grid spacing of the azimuth angle

  • precision (float) – Angular resolution of the solution

  • tol (float) – Allowed error in final ray-tracing position (m)

Return type

array (time of arrival, initial azimuth angle, initial takeoff angle)

Position(self, lat, lon, elev):

Defines a 3-D position in geographic coordinates, which can be easily converted to local coordinates with a given reference position.
